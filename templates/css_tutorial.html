CSS Basics

Learn how to style webpages using Cascading Style Sheets.

What is CSS?

CSS controls the appearance of your webpage — colors, fonts, spacing, alignment, animations, and layout.

Selector Types:

  • element selector: p { color: red; }
  • class selector: .box { padding: 10px; }
  • ID selector: #title { font-size: 2rem; }

Example CSS:

body {
    background: white;
    font-family: Arial;
}

h1 {
    color: navy;
}

button {
    padding: 10px;
    border-radius: 6px;
}

Connecting CSS to HTML:

<link rel="stylesheet" href="style.css">
