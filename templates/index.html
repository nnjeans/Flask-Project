How to Use Flask

  1. Install Flask: Run pip install Flask in your terminal.
  2. Create Your App: Set up your Python script with Flask routes.
  3. Render Templates: Use render_template() to serve HTML pages.
  4. Run Your Server: Execute your script to start the Flask development server.
  5. Access Your Site: Open http://localhost:5000 in your browser.

Learn Basic HTML & CSS

HTML (HyperText Markup Language) is used to structure content on the web, while CSS (Cascading Style Sheets) styles that content.

HTML Basics

  • Tags: Elements like <h1>, <p>, and <a>.
  • Attributes: Additional info inside tags, e.g., href in links.
  • Creating a Link: <a href="URL">Link Text</a>

CSS Basics

  • Selectors: Target elements, e.g., body, .class, #id.
  • Properties: Define styles like color, background-color, font-size.
  • Example:
    body { background-color: #e0f7fa; }

Start experimenting by editing your HTML and CSS to customize your website!
