HTML Basics

Understanding HTML

  • Tags: Elements like <h1>, <p>, <a>.
  • Attributes: Extra info inside tags (example: href in <a>).
  • Example Link: <a href="URL">Click Me</a>

Sample HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Sample Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
